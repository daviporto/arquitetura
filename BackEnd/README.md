# FindMe api

# How to install

```php
docker-compose up
```

```php
composer install
```

```php
php artisan key:generate
```

```php
php artisan jwt:secret
```

```php
php artisan migrate
```

### to populate the database 

```php
php artisan db:seed
```

### to link the public storage directory

```php
php artisan storage:link
```
